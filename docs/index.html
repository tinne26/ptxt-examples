ptxt interactive examples

This page links to a few interactive examples from ptxt, a pixel font text library made for Ebitengine, the open source 2D game engine written in Golang by Hajime Hoshi.

Examples have been slightly adapted and compiled to WASM so they can be executed directly on the web:

  • Words: the example on ptxt's README.md. Just words flashing by; nothing special about it.
  • Aligns: interactive program to see how different align modes and text directions operate.
  • Measure: text area and visualization for ptxt measuring.
  • Wrap: change the wrap point to see how basic line wrapping behaves in ptxt.

You can find the original sources at ptxt-examples/gpu/.
